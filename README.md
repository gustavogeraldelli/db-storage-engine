# About
This repository contains the assignments for the college's Information Organization and Retrieval course, where the study focuses on how data is organized and manipulated in secondary memory. 

The main objective of these assignments is to simulate how a database stores and manipulates data in secondary memory (disk), using efficient data structures to minimize the number of disk accesses.

The input files contains predefined SQL queries for both assignments.

## T01: Indexing
Implementation of a storage system using indexing (primary and secondary indexes).

## T02: B-Tree
Implementation of a storage system using B-Tree.
